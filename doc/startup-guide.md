# 区块链数据采集服务启动指南

## 1. 环境准备

### 1.1 基础环境要求
- Go 1.21+
- MySQL 8.0+
- Redis (可选，用于缓存)
- Node.js 18+ (前端开发)

### 1.2 系统要求
- CPU: 4核+
- 内存: 8GB+
- 磁盘: 100GB+ (取决于数据存储需求)
- 网络: 稳定的网络连接，建议带宽 10Mbps+
